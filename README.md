# Wemap Benchmarker

## Install

```bash
npm install -g kiss-cli
npm install
```

## DOM Manipulation benchmark

### Single suite

Create a new test suite in `suites` folder
```bash
kiss suites/<my_suite_name>.suite
```

Add new suite into `suites/_index.js` collection

(re)start NodeJS benchmark server
```bash
npm run start
```
